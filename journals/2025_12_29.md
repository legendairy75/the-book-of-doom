## symbolic links
	- `ln -s target name`
	- this command links a file or directory to the current directory
- ## archiving and compressing
- gzip and tar are the two most common ways to archive and compress files and directories
- ### gzip
	- (*GNU-Z*ip) is the current standard for Unix compression
	- a file ending in *.gz* is a GNU-Zip archive.
	- use `gunzip file.gz` to decompress a file
	- use `gzip file` to compress a file
- ### tar
	- *gzip* does not archive files, for that use **tar**
	- |flag|operation|
	  |---|---|
	  |c|creates archive|
	  |v|list output|
	  |f|file operation ( file name **must** come after)|
	  |x|unpack|
	  |t|list contense of archive|
	- use *-* after *f* to direct to  standard input or output
