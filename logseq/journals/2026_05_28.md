- # D2
  diagram scripting language that turns text into diagrams
	- Install
	  `curl -fsSL https://d2lang.com/install.sh | sh -s --``
	- uninstall
	  `curl -fsSL https://d2lang.com/install.sh | sh -s -- --uninstall`
	- ## Getting started
		- make a *.d2* file eg *input.d2*
		- paste this code
		  `x -> y: hello world`
		- exicute d2
		  `d2 input.d2`
		- should create input.svg
