# AWS
	- ## EC2
		- to ssh into ec2 go to directory that containe the .pem and runb
		  `chmod 400 [ssh].pem`
		- the user name is usualy the os name unless you are using *amazon linux* in which case the username is **ec2-user**
		- you will also need the public facing ip address
			- e2c > instances > instance > detais > public ipv4 address
		- `ssh -i "[ssh].pem ec2-user@[public-ip]`
		- ### amazon linux
			- to install docker on amazon linux:
			  ``` sh
			  # update registry
			  sudo yum update -y
			  # install docker
			  sudo yum install -y docker
			  # start docker
			  sudo service docker start
			  # if you want to start docker after boot
			  sudo systemctl enable docker
			  # add user to docker groupe
			  sudo usermod -a -G docker ec2-user
			  ```
	- ### for auto deploy
		- in a github actions to auto deploy to ec2 go to repo secrets, add **repo secret** and add the host ip ad one secret and the *contence* of the .pem file to another secret
