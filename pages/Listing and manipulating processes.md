- Each process has a numeric *process ID (PID)*
- to list pid type `ps`
- |Field|Desc|
  |---|---|
  |PID|the process id|
  |TTY|terminal where process is running|
  |STAT|process status(S means sleeping while R means running|
  |TIME|The amount of CPU time The process has used|
  |CMD|The command used to run the program|
